# MiMo Agent Workflow
基于小米MiMo V2.5 Pro的多智能体自动化工作流系统，支持代码评审、文档解析、数据批量处理。

## 项目介绍
本项目基于LangGraph+Dify构建多节点Agent工作流，实现：
- 任务自动拆解与规划
- 多轮代码执行与结果校验
- 长上下文文档结构化解析
- 企业级知识库批量入库

## 技术栈
- 大模型：小米MiMo V2.5 Pro / GPT-4o
- 框架：LangGraph, LangChain, Dify
- 语言：Python 3.10+
- 部署：Docker + FastAPI

## 使用场景与Token消耗
| 场景 | 单任务平均轮次 | 单次Token消耗 | 日均调用量 |
|------|----------------|--------------|------------|
| 代码评审 | 12-15轮 | 20-30万 | 15次 |
| 文档解析 | 5-8轮 | 50-80万 | 8次 |
| 数据处理 | 8-10轮 | 10-15万 | 20次 |
日均总Token消耗：约300万Token
